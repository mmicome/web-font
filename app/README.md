创建自己的cli 工具， 支持安装到客户端， 如 vue init + 模板文件（git远程拉取骨架）

设计原则
形参设置一定语义化， 且尽可能全文统一， 尤其是涉及数据库字段

导入导出， 接口返回， 最好做字段映射， 即使字段一致

前端字段常量一定统一维护， 不直接写属性字段， 即 obj.prop 方式 而使用 obj[CONST.FIELD.PROP] 易于维护
