Info Tip Doc

1. 引入文件


<script src="ploymer.js"></script>  //如果兼容IE则需引入，且动态加载，不会消耗资源
<script src="tip.js"></script>           //tip 组件

2. 使用 tip

2.1 simple tip: 使用 content 属性定义简单提示信息


<x-tip content="test a info tip lang lang tip is wrap "></x-tip>

2.2 custom tip: 使用模板自定义内容，完全自由 ; 


<x-tip>
    <article slot="my-tip">
        <header>test a info tip</header>
        <img src="http://img.hb.aicdn.com/56e32a3058f50f5fb364a86bf4a7b4af7e17c1be353a7-5VUnTc_fw658" alt="" />
    </article>
</x-tip>
attention： 模板最外层元素包含属性slot="my-tip"

2.3 url tip: 通过url 引入外部独立文档(感觉不需要)

2.4 ajax tip：动态创建提示信息，比如 ajax data

simple

document.querySelector("x-tip").setAttribute('content', 'simple 方式');
custom

$.get(url, function(data) {
    var div = document.createElement('div');
    div.setAttribute('slot', 'my-tip');
    div.style.width = "100px";
    div.textContent = data;
    document.querySelector("x-tip").appendChild(div);
})
